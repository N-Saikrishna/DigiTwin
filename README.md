# Academic Digital Twin

A Flask-based web application that simulates a student's academic trajectory using a personalized "Digital Twin" model.

## Features

- Academic risk prediction
- GPA trajectory simulation
- Burnout probability estimation
- Scenario-based recommendations

## Tech Stack

- Python
- Flask
- HTML / CSS
- Jinja Templates

## Installation

Clone the repo:

git clone https://github.com/yourusername/academic-digital-twin.git

Install dependencies:

pip install -r requirements.txt

Run the app:

python app.py

Open in browser:

http://127.0.0.1:5000
